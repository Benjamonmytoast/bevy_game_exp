# Bevy 2D Game Exploration

This project is an exploration of creating a simple 2D game using [Bevy](https://bevyengine.org/), a data-driven game engine built in Rust.

## Overview

A hands-on experiment with Bevy's core concepts including:
- Entity-Component-System (ECS) architecture
- 2D rendering and sprite handling
- Game loop and update systems
- Input handling
- Basic game mechanics

## Getting Started

### Prerequisites
- Rust (latest stable)
- Cargo

### Running the Project

```bash
cargo run
```

## Project Structure

- `src/` - Main source code
- `assets/` - Game sprites and resources

## Learning Goals

- Understand Bevy's ECS pattern
- Build a functional 2D game prototype
- Explore rendering and physics in Bevy
